<h1>Layout 5</h1>
<br>
<p>Exibe o menu na parte superior do site no lado esquerdo do cabeçalho. O aside com o conteúdo relacionado fica à esquerda do main que tem o conteúdo principal do site. Quando apresentado em telas menores, como smartphones, o menu fica oculto, sendo exibido apenas quando o botão de menu no canto superior direito é clicado.</p>
